== Description ==
Syntax Highlighter is a Wordpress plugin allowing you to beautifully highlight code snippets embedded to your posts and pages. Works with Gutenberg and TinyMCE editors.

**Features**
* 8 built in themes
* 150+ different programming languages
* Extends default Gutenberg code block
* Works with TinyMCE

== Installation ==
1. Upload the plugin files to the \'/wp-content/plugins/pmd-syntax-highlighter/\' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Use the \'Settings->Syntax highlighter\' screen to configure the plugin

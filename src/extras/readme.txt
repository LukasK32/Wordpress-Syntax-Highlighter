== Description ==
Syntax highlighter is free, easy to use, Wordpress plugin extending default Gutenberg code block by adding code highlighting for all languages supported by Prism.js.

== Installation ==
1. Upload the plugin files to the \'/wp-content/plugins/pmd-syntax-highlighter/\' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Use the \'Settings->Syntax highlighter\' screen to configure the plugin
